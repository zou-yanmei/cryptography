 流密码多次填充攻击实验报告
 一、基础信息
- 编程语言：Python
- 实验目标：破解11段使用同一密钥流加密的密文，完成第11段目标密文的解密

 二、核心实现思路
1. 消去密钥流：将多段密文两两逐字节XOR，利用公式 `C₁^C₂ = M₁^M₂` 直接消除密钥流，得到明文异或结果；
2. 推断明文字符：利用空格(0x20)与英文字母XOR会大小写互换的核心规律，精准判断明文中的空格位置；
3. 还原密钥流：根据空格位置计算密钥字节（密钥 = 密文字节 ^ 0x20），逐位还原完整密钥流；
4. 解密目标密文：用还原的密钥流与目标密文XOR，最终得到明文。

 三、分析方法说明
使用流密码密钥重用攻击（多次填充攻击）+ 英文空格特征统计分析法：
1. 遍历所有密文组合，逐位异或并统计符合「空格^字母」特征的次数；
2. 以统计得分最高的位置作为可信空格位置，计算对应密钥字节；
3. 该方法无需知晓原始密钥，仅依靠密文特征和英文文本规律即可完成破解。

 四、明文内容确认方法
1. 密钥流验证：通过多密文的空格置信度投票生成密钥流，保证密钥字节的准确性；
2. 解密规则：严格遵循流密码解密公式 `明文 = 密文 ^ 密钥流`，仅保留可打印ASCII字符；
3. 语义验证：解密结果为连贯的英文语句，符合自然语言逻辑，可确认明文真实有效。

 五、解密得到的明文
The secuet message is: Whtn using aastream cipher, never use the key more than once

 六、实验总结
本次实验验证了流密码的核心安全漏洞：密钥流绝对不能重复使用，一旦重用，攻击者可通过密文异或+字符特征分析，轻松破解密文；基于空格特征的统计分析法，是破解流密码密钥重用漏洞的经典且高效的方法。
