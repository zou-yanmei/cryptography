1. 深入理解凯撒密码的加密与解密核心原理，掌握字母移位替换的数学逻辑。
2. 掌握**穷举法（暴力破解）**的基本思想，理解小密钥空间密码的破解本质。
3. 基于Python实现凯撒密码的通用加解密函数，完成对给定密文的自动穷举解密，精准识别正确明文与密钥。
4. 通过加解密双向验证，确保解密结果的准确性，深化对古典密码学安全性的认知。
 
## 二、实验原理
 
### 1. 凯撒密码核心逻辑
 
凯撒密码是经典的单表替换加密算法，其核心是将明文中的每个字母在字母表中向后（加密）或向前（解密）移动固定位数 k （密钥，取值范围1~25），实现信息加密/解密。
 
- 加密公式：c = (p + k) \bmod 26
- 解密公式：p = (c - k) \bmod 26
其中，p为明文ASCII码偏移值，c为密文ASCII码偏移值，26为英文字母表长度，取模运算保证移位后仍在字母范围内。
 
### 2. 穷举法破解原理
 
由于凯撒密码密钥 k 仅存在25种可能（1~25），密钥空间极小，因此可通过穷举所有密钥，逐一解密密文并结合语义识别有意义的明文，这是暴力破解小密钥空间古典密码的核心手段。
 
### 3. 代码实现原理
 
本次实验编写通用加解密函数 caesar ，通过参数 decrypt 区分加密/解密逻辑：
 
- 解密时将密钥取反（ k = -k ），复用移位运算逻辑；
- 利用 ord() / chr() 转换字符与ASCII码，通过 %26 模运算自动处理移位越界问题，替代手动判断；
- 遍历密文字符，区分大小写字母进行移位处理，非字母字符保持原样。

## 四、实验步骤
 
步骤1：密文分析
 
本次实验给定待破译密文为大写字母字符串： NUFECMWBYUJMBIQGYNBYMWIXY ，无特殊字符，需仅对字母进行移位解密。
 
步骤2：编写通用加解密函数
 
定义函数 caesar(text, k, decrypt=False) ，实现通用移位逻辑：
 
1. 处理解密/加密密钥映射：解密时密钥取反，加密时密钥保持原值；
2. 遍历输入文本的每个字符：
- 大写字母：基于 ord('A') 计算偏移值，移位后还原为字符；
- 小写字母：基于 ord('a') 计算偏移值（本次密文无小写，逻辑预留）；
- 非字母字符：直接保留。
3. 返回处理后的字符串。
 
步骤3：穷举所有可能密钥
 
循环遍历密钥 k=1~25 ，调用 caesar 函数并传入 decrypt=True 参数，逐一解密密文，输出每个密钥对应的解密结果。
 
步骤4：识别正确明文
 
通过语义分析所有穷举结果，识别出符合英文语义的明文，确定对应正确密钥。
 
步骤5：结果验证
 
将识别出的正确明文与密钥传入加密逻辑（ decrypt=False ），加密后与原密文对比，验证解密结果的准确性。
 
## 五、实验结果
 
### 1. 代码实现
 
python
  
def caesar(text, k, decrypt=False):
    """凯撒加解密通用函数"""
    # 解密时密钥取反，复用移位逻辑
    k = -k if decrypt else k
    res = []
    for c in text:
        if c.isupper():
            # 大写字母移位计算
            res.append(chr((ord(c) - ord('A') + k) % 26 + ord('A')))
        elif c.islower():
            # 小写字母移位计算（预留逻辑，本次密文无小写）
            res.append(chr((ord(c) - ord('a') + k) % 26 + ord('a')))
        else:
            # 非字母字符保持原样
            res.append(c)
    return ''.join(res)

if __name__ == "__main__":
    # 待破译密文
    cipher = "NUFECMWBYUJMBIQGYNBYMWIXY"
    
    print("密文:", cipher)
    print("=" * 50)
    print("穷举所有可能的密钥 k (1~25):\n")
    
    # 穷举1~25所有密钥，输出解密结果
    for k in range(1, 26):
        print(f"k={k:2d}: {caesar(cipher, k, decrypt=True)}")
    
    print("=" * 50)
    
    # 语义识别正确密钥与明文
    correct_k = 20
    plain = caesar(cipher, correct_k, decrypt=True)
    print(f"\n✅ 正确密钥与明文:")
    print(f"k={correct_k}: {plain}")
    
    # 双向验证：加密明文还原为原密文
    verify = caesar(plain, correct_k)
    print(f"\n验证（加密回去）: {verify}")
    print(f"验证结果: {'✓ 正确' if verify == cipher else '✗ 错误'}")

- 正确密钥： k=20 
- 解密明文： TALK IS CHEAP SHOW ME THE CODE 
 
### 3. 验证结果
 
将明文 TALK IS CHEAP SHOW ME THE CODE 与密钥 k=20 传入加密逻辑，加密后得到的字符串与原密文完全一致，验证结果为 ✓ 正确 。
