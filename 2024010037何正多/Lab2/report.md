# 流密码密钥重用攻击实验报告

## 一、实验目的

本实验旨在通过分析多段使用相同密钥加密的密文，利用流密码密钥重用的漏洞，推导出密钥并解密目标密文。

## 二、实验方法

1. 将多段密文两两 XOR，消去密钥流，得到明文 XOR 明文。
2. 利用空格与字母 XOR 的规律推断各位置的明文字符。
3. 逐步还原密钥流，最终解密目标密文。

## 三、实验分析

### 1. 使用的分析方法

- **XOR 消元**：当相同的密钥 K 用于加密多条明文 P_i 时，有：
  C_i = P_i ⊕ K
  C_i ⊕ C_j = (P_i ⊕ K) ⊕ (P_j ⊕ K) = P_i ⊕ P_j
- **空格识别**：空格字符（0x20）与字母异或会产生大小写转换：
  - 大写字母(65-90) ⊕ 0x20 = 小写字母(97-122)
  - 小写字母(97-122) ⊕ 0x20 = 大写字母(65-90)
- **密钥推导**：一旦确定某个位置是空格，可以直接推导密钥：
  K[j] = C_i[j] ⊕ 0x20

### 2. 如何确认目标密文的明文内容

1. **数据预处理**：将11条十六进制密文转换为字节流，并统一长度（用0填充较短的密文）。
2. **空格识别**：对所有密文对进行逐字节异或，如果异或结果是字母，则其中一个字节很可能是空格。
3. **密钥推导**：从已识别的空格推导密钥。
4. **解密**：用推导的密钥填充明文中缺失的部分，仅保留可打印 ASCII 字符。
5. **修正误差**：手动修正识别误差，得到最终正确明文。

### 3. 解密得到的明文是什么

最终解密得到的明文是：

> The secret message is: when using a stream cipher, never use the key more than once

## 四、实验结论

通过本次实验，我们成功利用流密码密钥重用的漏洞，推导出密钥并解密了目标密文。实验结果表明，流密码的安全性高度依赖于密钥的唯一性，密钥重用会导致严重的安全隐患。
