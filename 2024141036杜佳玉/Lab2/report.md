实验分析说明
1. 采用的分析方法
多密文统计破解法（空格辅助验证）：
核心原理：同一密钥加密 → C1^C2 = M1^M2，密钥流被完全消除；
破解思路：遍历每个位置的所有可能密钥字节（0-255），用候选密钥解密其余 10 段密文的同一位置；
筛选规则：能让最多明文为可打印 ASCII 字符（空格、字母、数字、符号）的字节，就是真实密钥字节；
辅助特性：英文文本空格极多，空格 (0x20) 与字母异或会翻转大小写，大幅提升破解准确率。
2. 明文确认依据
还原的密钥流能解密所有 11 段密文为通顺的英文句子；
目标密文解密结果为完整、语义通顺的英文文本，无乱码；
所有位置的密钥字节都满足「最多可打印字符」的最优条件，无冲突。

The secret message is: When using a stream cipher, never use the key more than once
实验总结
核心漏洞：流密码绝对不能重复使用密钥，否则攻击者可轻松破解明文；
攻击本质：利用多密文异或消除密钥流，结合英文文本特征还原明文；
关键技巧：可打印字符统计 + 空格大小写翻转规律，是流密码破解的核心手段；
安全启示：实际应用中，流密码必须配合一次性密钥（如 OTP）或安全的密钥协商机制。
