1. 实验目的
理解流密码的加密原理与安全缺陷
掌握 **多次填充攻击（Many-Time Pad Attack）** 的核心方法
利用字符统计规律还原密钥流，解密目标密文
2. 实验原理
流密码加密公式：
C=M⊕PRG(k)
若同一密钥k被多次使用，则：
C1​⊕C2​=(M1​⊕PRG(k))⊕(M2​⊕PRG(k))=M1​⊕M2
关键特征：空格字符（0x20）与英文字母异或时，会翻转大小写（A ↔ a），利用此规律可推断明文中的空格位置，进而反推密钥流。
3. 实验步骤
数据准备：将 11 段十六进制密文转换为字节数组
密钥流还原：
  遍历每个字节位置，假设该位置为空格，计算候选密钥字节
  统计候选密钥解密后为字母的次数，票数最高者为正确密钥
解密目标密文：用还原的密钥流与目标密文异或，得到明文
结果验证：检查明文是否为通顺的英文语句
4. 实验结果
运行代码后，得到正确明文：
The secret message is: When using a stream cipher, never use the key more than once
该明文语义完整，符合流密码安全警示，验证攻击成功。
5. 实验分析与总结
  核心漏洞：流密码绝不能复用密钥，否则会被多次填充攻击彻底破解
  攻击关键：利用空格与字母的异或特征，通过统计投票还原密钥流
  安全启示：实际应用中必须保证密钥的一次性使用（One-Time Pad），或使用安全的密钥协商机制
