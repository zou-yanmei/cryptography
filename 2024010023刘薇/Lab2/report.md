# 一、攻击方法

本次实验采用流密码多次重用密钥攻击（多次填充攻击）。流密码加密公式为：  
$C_i = M_i \oplus PRG(k)$  
由于所有密文均使用同一密钥流加密，因此任意两条密文异或时，密钥流会被抵消，得到：  
$C_1 \oplus C_2 = M_1 \oplus M_2$。  

利用英文文本中空格字符（`0x20`）出现频率极高，且空格与字母异或会切换大小写这一典型特征，对明文结构进行统计推断，从而还原出密钥流，最终解密目标密文。

# 二、密钥流推断过程

1. **数据预处理**  
   将给出的 11 条十六进制密文全部转换为字节数组，前 10 条用于攻击分析，最后 1 条为待解密目标密文。

2. **逐字节位置分析**  
   对密钥流每一个字节位置，遍历前 10 条密文，进行两两异或，得到对应位置的明文异或值 $M_1 \oplus M_2$。

3. **空格特征匹配**  
   若该异或结果为大小写字母（A-Z、a-z），则说明该位置极大概率其中一条明文为空格。利用公式反向推导可能的密钥字节：  
   $k = C_i \oplus 0x20$。

4. **统计投票确定密钥**  
   对每个位置收集所有候选密钥字节，选择出现次数最多的字节作为该位置的最终密钥，保证结果最可靠。

5. **无特征位置补充处理**  
   对少数无明显字母特征的位置，额外筛选能解密出可打印 ASCII 字符的密钥字节，确保明文可读。

6. **组合完整密钥流**  
   将所有位置确定的密钥字节拼接，得到完整密钥流。

# 三、解密与最终明文

使用推导得到的密钥流与目标密文逐字节异或，完成解密：  
$M_{\text{target}} = C_{\text{target}} \oplus \text{key\_stream}$  

最终得到完整可读明文：  
The secret message is: when using a stream cipher, never use the key more than once
