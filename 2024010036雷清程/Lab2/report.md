多次填充攻击流密码实验报告

本次实验采用多次填充攻击（Many Time Pad）分析方法，核心利用同一密钥加密多段密文的安全漏洞：


1. 空格特征统计：英文文本中空格（ASCII 32）高频出现，空格与字母异或结果仍为字母，以此推断空格位置

2. 密文异或特性：两段同密钥加密的密文异或后，密钥流被消除，仅保留明文异或结果

3. 密钥还原：通过空格位置计算对应密钥字节，逐步推导完整密钥流

4. 密文解密：用还原密钥与目标密文异或，得到明文

三、明文内容确认方式

1. 字符有效性验证：解密结果中可打印字符（32-126 ASCII）占比高，无大量乱码

2. 语义连贯性：明文为完整英文语句，符合自然语言逻辑

3. 密钥一致性：多段密文推导的密钥字节无冲突，验证密钥还原准确性

解密明文结果

目标密文解密后明文：
The secret message is: When using a stream cipher, never use the key more than once

实验总结

1. 通过本次实验，熟练掌握了流密码XOR加密的基本原理，深入理解了异或运算的可逆性及核心特性，明确了多次填充攻击的实施逻辑与关键步骤。

2. 深刻认识到流密码的安全前提——同一密钥不可重复使用，同一密钥重复加密多段明文会产生严重安全漏洞，易被攻击者利用明文统计特征还原密钥与明文。

3. 成功实现了基于英文空格特征的密钥还原与密文解密，通过多维度验证，充分验证了多次填充攻击方法的有效性与可行性。

4. 在实验过程中，提升了Python处理字节数据、实现异或运算、分析文本特征的实践能力，进一步巩固了密码学的基础理论知识。
