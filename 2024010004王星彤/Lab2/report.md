实验报告

一、实验原理

流密码加密：C = P \oplus K
同一密钥加密多条明文时：
C_i \oplus C_j = P_i \oplus P_j
利用英文文本特征（空格、字母高频），可逐位反推密钥流 K。

二、实验方法

1. 将11段十六进制密文转为字节流。

2. 密文两两异或，结合英文可打印字符规律猜测明文片段。

3. 由 K = C \oplus P 逐位推导完整密钥流。

4. 用密钥流与目标密文异或，得到明文。

三、密钥流推断过程

1. 对齐所有密文长度，两两异或去除密钥流影响。

2. 利用高频空格（0x20）与字母异或特征，锁定局部明文字符。

3. 反算对应密钥流字节，逐步扩展至完整密钥流。

4. 用其他密文验证密钥流一致性，确保无误。

四、解密结果

解密公式：P_{target} = C_{target} \oplus K
最终明文：
The secret message is: when using a stream cipher, never use the key more than once

五、结论

流密码密钥复用存在严重安全漏洞，攻击者可通过多密文攻击完全恢复明文与密钥流。
