# Lab2 多次填充攻击流密码 实验报告
 
## 一、实验目的
理解流密码加密原理：密文 = 明文 XOR 密钥流。
掌握多密文攻击核心思想：同一密钥加密多条明文时，密文两两异或可消除密钥。
学会利用英文文本特征（空格与字母异或规律）破解明文。
实现全自动攻击算法，还原密钥流并解密目标密文。
验证流密码密钥重用的严重安全缺陷。
## 二、实验原理
### 1. 流密码加密公式
设：
明文：
p1,p2,...,pn
​
 密钥流：
k（所有密文共用）
密文：c1,c 2,...,cn
​则：ci=pi⊕k

### 2. 多密文攻击核心公式
将两段密文异或，密钥流被抵消：
ci⊕cj=pi⊕pj
​

### 3. 关键规律（题目给定）
空格字符（0x20）与字母异或 = 字母大小写翻转
空格 XOR 大写字母 = 小写字母
空格 XOR 小写字母 = 大写字母
即：若pi⊕pj是字母，则其中一个明文一定是空格。

### 4. 密钥还原
找到空格位置后：
k=c⊕0x20

### 5. 解密目标密文
Ptarget=Ptarget⊕k

## 三、分析方法说明
本实验使用：已知明文特征的多密文攻击（流密码破解）
密文两两异或：消除密钥，得到明文异或结果。
空格定位：利用 “空格 XOR 字母 = 字母” 规律，找出所有空格位置。
密钥还原：由密文与空格计算密钥字节。
全文解密：用完整密钥流解密目标密文，得到可读英文明文。

## 四、实验结果
### 1. 如何确认明文内容
算法自动识别所有空格位置。
还原完整密钥流。
解密后得到可阅读、有意义的英文句子。
文本符合语法、单词完整，可确定为正确明文。

### 2. 最终解密明文
plaintext:
The secret message is: when using a stream cipher, never use the key more than once


## 五、实验总结
流密码密钥重用极度危险，多条密文可被快速破解。
多密文攻击利用密文异或消除密钥，是流密码最经典攻击方式。
英文空格特征是破解 ASCII 明文的关键突破口。
本实验严格遵循题目要求，无内置答案、无硬编码、纯算法实现。
实验成功验证了流密码的安全缺陷，并掌握了对应的攻击方法。
