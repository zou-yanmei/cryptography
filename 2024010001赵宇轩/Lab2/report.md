多段密文异或攻击实验报告

一、实验目的

本实验旨在通过多时间垫密码攻击方法，利用多段使用相同密钥流加密的密文，通过两两异或消去密钥流，结合空格与字母异或的统计规律，逐步还原明文并解密目标密文。

二、核心思路

1. 异或消去密钥流
      由于所有密文使用相同的密钥流加密，满足：
      Cᵢ = Pᵢ ⊕ K
      因此两两密文异或可得：
      Cᵢ ⊕ Cⱼ = (Pᵢ ⊕ K) ⊕ (Pⱼ ⊕ K) = Pᵢ ⊕ Pⱼ
      消去了密钥流 K，得到明文之间的异或结果。
2. 利用空格与字母的异或规律
      英文文本中空格（0x20）出现频率极高。空格与字母异或的特性为：
   · 空格(0x20) ⊕ 大写字母(0x41–0x5A) = 小写字母(0x61–0x7A)
   · 空格(0x20) ⊕ 小写字母(0x61–0x7A) = 大写字母(0x41–0x5A)
        因此，若 Pᵢ ⊕ Pⱼ 的结果为字母，则说明 Pᵢ 和 Pⱼ 中有一个是空格。
3. 逐步还原明文与密钥流
   · 通过两两密文异或，识别出各位置的空格。
   · 利用空格（0x20）与密文异或，推导出对应位置的密钥流：K = Cᵢ ⊕ 0x20。
   · 用推导的密钥流解密其他密文，结合可打印 ASCII 字符范围（0x20–0x7E）验证明文。

三、实验步骤

1. 数据预处理
   · 将所有密文从十六进制转换为字节流。
   · 统一密文长度（不足补0），便于逐字节分析。
2. 两两异或识别空格
   · 遍历所有密文对 (Cᵢ, Cⱼ)，计算 Cᵢ ⊕ Cⱼ。
   · 若异或结果为字母（65–90 或 97–122），则标记 Cᵢ 或 Cⱼ 对应位置为空格（0x20）。
3. 推导密钥流
   · 对每个位置，若某密文被标记为空格，则计算密钥流：K = Cᵢ ⊕ 0x20。
4. 解密目标密文
   · 使用推导的密钥流解密目标密文（最后一个密文）。
   · 对无法推导的位置，标记为“？”，并通过手动修正补全。

四、分析方法

· 统计分析法：利用英文文本中空格的频率特性，结合异或运算的数学性质。
· 已知明文攻击：通过识别空格（已知明文）推导密钥流。
· 暴力验证：对推导的明文进行可打印 ASCII 字符范围验证。

五、实验结果

1. 目标密文明文确认
   · 通过密钥流解密得到初步结果，结合英文语义进行手动修正。
   · 修正字典基于常见英文单词和语法结构（如 "The secret message is:"）。
2. 解密得到的明文

The secret message is: when using a stream cipher, never use the key more than once.

六、结论

本实验成功利用多时间垫密码攻击，通过两两密文异或消去密钥流，结合空格与字母的异或规律，逐步还原了明文并解密了目标密文。实验验证了密钥流重用在多表代换密码中的严重安全隐患，强调了一次性垫密码中密钥流必须唯一且随机的重要性。
