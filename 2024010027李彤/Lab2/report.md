一、实验目的
通过分析多段使用同一密钥流加密的流密码密文，利用异或特性还原密钥流，最终解密目标密文，验证流密码 “同一密钥绝不能重复使用” 的安全准则。

二、实验环境

编程语言：Python 3.10+
工具：VS Code / PyCharm
输入：11 段使用同一密钥流加密的十六进制密文


三、核心原理与分析方法

1. 流密码加密公式
流密码通过将明文与密钥流进行异或运算实现加密：Ci​=Mi​⊕PRG(k)其中 Ci​ 为密文，Mi​ 为明文，PRG(k) 为密钥流。
2. 多次一钥攻击原理
若多段密文使用同一密钥流加密，则：C1​C2​​=M1​⊕PRG(k)=M2​⊕PRG(k)​将两段密文异或，密钥流被消除：C1​⊕C2​=M1​⊕M2​攻击者无需知道密钥，即可直接分析明文异或结果。
3. 空格特征推断明文
英文文本中空格字符（0x20） 与英文字母（[a-zA-Z]）进行异或时，会翻转字母的大小写：

大写字母 ↔ 小写字母
若 Ci​⊕Cj​=0x20，则其中一个明文字符必为空格

4. 密钥流还原方法
通过统计所有密文对中异或结果为 0x20 的位置，反推候选密钥流字节，选择出现次数最多的候选值作为有效密钥流，最终还原完整密钥流。

四、实验结果与分析

1. 分析方法说明
分析方法：多次一钥流密码攻击，基于 “同一密钥流加密的密文两两异或可消除密钥流” 的特性，结合英文空格与字母异或的大小写翻转规律，还原密钥流并解密。
明文确认方法：
还原的密钥流解密后所有字节均为可打印 ASCII 字符（字母、空格、标点），符合英文文本特征。
解密结果为连贯英文句子，语义通顺，符合流密码安全警示的实验背景。
多段密文解密后均为可读文本，验证了密钥流还原的正确性。

2. 最终明文结果

The secret message is : when using a stream cipher, never use the key more than once.


五、实验结论

本实验成功实现了流密码多次一钥攻击，验证了流密码 “同一密钥绝不能重复使用” 的安全准则。
利用空格与字母异或的特征，可有效推断明文字符并还原密钥流，最终完全解密目标密文。
实验结果表明：流密码若重复使用同一密钥，攻击者可无需知晓密钥本身，直接通过密文异或分析还原全部明文，存在严重安全隐患。

六、实验总结

本次实验深入理解了流密码的工作原理与安全漏洞，掌握了基于空格特征的多次一钥攻击方法，通过代码实现了密钥流还原与密文解密，强化了对密码学安全原则的认识。
